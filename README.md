# EchonX

AI-powered audio social platform that transforms social feeds into personalized podcast experiences.

## Overview

EchonX converts noisy social media feeds into calm, high-signal audio experiences.

Instead of endless doomscrolling, users can listen to personalized AI-generated audio feeds based on the accounts and topics they follow.

The platform focuses on reducing information overload while improving content quality and user focus.

## Core Features

- AI-generated audio feeds
- Personalized podcast experience
- Semantic content ranking
- Social content aggregation
- Responsive cross-device interface
- Subscription system with Stripe
- Real-time content processing
- Creator-focused tools

## Tech Stack

- Next.js
- TypeScript
- React
- Supabase
- Stripe
- Vercel
- AI APIs
- TailwindCSS

# Explicações em HTML (EchonX) — temporário

## URL principal (insights)

Com `npm run dev` (porta **3002**):

**http://localhost:3002/docs/html/readme**

Página no app (cards + resumos). Os HTML completos ficam em `/docs/html/*.html`.

| URL | Uso |
|-----|-----|
| `/docs/html/readme` | Hub temporário (recomendado) |
| `/docs/html/echonx-explicacoes-index.html` | Índice HTML simples |
| `/docs/explicacoes` | Redireciona para `/docs/html/readme` |

Arquivos estáticos: editar **`public/docs/html/`** (o app serve essa pasta).

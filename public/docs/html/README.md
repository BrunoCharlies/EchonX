# Explicações em HTML (EchonX)

Páginas HTML autocontidas para documentar decisões, brainstorms e análises técnicas discutidas com o assistente.

## Convenção

- Um arquivo por tema: `AAAA-MM-assunto-curto.html`
- Índice central: [echonx-explicacoes-index.html](./echonx-explicacoes-index.html)
- Abrir no navegador (duplo clique ou `file://`) — não exige servidor
- Estilo escuro alinhado ao produto EchonX

## Adicionar nova página

1. Criar o `.html` nesta pasta
2. Registrar no índice `echonx-explicacoes-index.html`
